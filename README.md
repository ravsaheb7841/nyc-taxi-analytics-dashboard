An end-to-end **Business Intelligence** project that analyzes the **NYC Yellow Taxi Trips 2015** dataset using **Google BigQuery (SQL)** and presents business insights through an interactive **Looker Studio Dashboard**.

The project demonstrates the complete analytics workflow—from raw data exploration and SQL analysis to dashboard development and business reporting.

---

# 📌 Project Overview

The objective of this project is to analyze taxi trip data and answer important business questions related to customer demand, revenue performance, payment behavior, vendor comparison, passenger trends, and trip distance.

Instead of creating visualizations directly from raw data, SQL was first used to perform analytical calculations. The validated results were then visualized in Looker Studio to create an interactive business dashboard.

---

# 🎯 Business Objectives

- Analyze taxi trip data using SQL
- Identify peak operating hours
- Monitor monthly revenue trends
- Compare vendor performance
- Analyze customer payment preferences
- Study passenger behavior
- Evaluate trip distance patterns
- Build an interactive dashboard for business users

# 📂 Dataset

**Dataset Name:** NYC Yellow Taxi Trips 2015

**Source:** Google BigQuery Public Dataset

The dataset contains taxi trip records including:

- Pickup & Drop-off Date/Time
- Trip Distance
- Fare Amount
- Total Amount
- Passenger Count
- Payment Type
- Vendor ID
- Pickup Coordinates
